================================================================================

- Create ensemble (orchestra and jazz band)
- Add musicians to both ensemble types
- Show ensemble details
- List all ensembles
- Switch between ensembles
- Modify musician's instrument
- Delete musician
- Change ensemble name
- View history
- Undo operations with ensemble context switching
- Redo operations

Expected Test Flow:
--------------------------------------------------------------------------------

1. Create orchestra E001 "Symphony Orchestra"
   - Should display: "Orchestra ensemble is created."
   - Should display: "Current ensemble is changed to E001."

2. Add violinist m001 (Ludwig van Beethoven)
   - Should display: "Musician is added."

3. Add cellist m002 (Yo-Yo Ma)
   - Should display: "Musician is added."

4. Show current ensemble
   - Should display orchestra with 1 violinist and 1 cellist

5. Create jazz band E200 "Modern Jazz Quartet"
   - Should display: "Jazz band ensemble is created."
   - Should display: "Current ensemble is changed to E200."

6. Add pianist m201 (Oscar Peterson)
   - Should display: "Musician is added."

7. Add saxophonist m202 (Charlie Parker)
   - Should display: "Musician is added."

8. Add drummer m203 (Max Roach)
   - Should display: "Musician is added."

9. Show current ensemble
   - Should display jazz band with pianist, saxophonist, and drummer

10. List all ensembles
    - Should display both E001 and E200

11. Switch to E001
    - Should change current ensemble to E001

12. Add violinist m003 (Itzhak Perlman)
    - Should display: "Musician is added."

13. Show ensemble
    - Should display 2 violinists and 1 cellist

14. Modify m001 to cellist
    - Should display: "Instrument is updated."

15. Show ensemble
    - Should display 1 violinist and 2 cellists

16. List history
    - Should show all undoable commands in undo list
    - Redo list should be empty

17. Undo modify instrument
    - Should display: "Command (Modify musician's instrument, m001, cellist) is undone."
    - m001 should be back to violinist

18. Show ensemble
    - Should display 2 violinists and 1 cellist again

19. Undo add musician m003
    - Should display: "Command (Add musician, m003, Itzhak Perlman, violinist) is undone."

20. Show ensemble
    - Should display 1 violinist and 1 cellist

21. Undo add drummer (context switches to E200)
    - Should display: "Command (Add musician, m203, Max Roach, drummer) is undone."
    - Should display: "The current ensemble is changed to E200 Modern Jazz Quartet."

22. Switch to E200 (already current)
    - Should remain at E200

23. Show ensemble
    - Should display jazz band with pianist and saxophonist, drummer = NIL

24. Undo add saxophonist
    - Should display: "Command (Add musician, m202, Charlie Parker, saxophonist) is undone."

25. Show ensemble
    - Should display only pianist, saxophonist = NIL, drummer = NIL

26. List history
    - Undo list should have fewer items
    - Redo list should have the undone commands

27. Redo add saxophonist
    - Should display: "Command (Add musician, m202, Charlie Parker, saxophonist) is redone."

28. Show ensemble
    - Should display pianist and saxophonist again

29. Redo add drummer
    - Should display: "Command (Add musician, m203, Max Roach, drummer) is redone."

30. Show ensemble
    - Should display all three musicians

31. Switch to E001
    - Should change current ensemble to E001

32. Show ensemble
    - Should display 1 violinist and 1 cellist (m003 was undone)

33. Delete musician m002
    - Should display: "Musician is deleted."

34. Show ensemble
    - Should display 1 violinist, cellist = NIL

35. Change ensemble name to "Vienna Philharmonic"
    - Should display: "Ensemble's name is updated."

36. Show ensemble
    - Should display new name

37. List history
    - Should show all commands including delete and rename

38. Undo rename
    - Should restore name to "Symphony Orchestra"

39. Show ensemble
    - Should display original name

40. Undo delete musician
    - Should restore m002

41. Show ensemble
    - Should display 1 violinist and 1 cellist again

42. Switch to E200

43. Rename to "Blue Note All-Stars"
    - Should display: "Ensemble's name is updated."

44. Show ensemble
    - Should display new jazz band name

45. List all ensembles
    - Should display both ensembles with current names

46. List history
    - Should show complete undo/redo lists

47. Exit system
    - Program should terminate
